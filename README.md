# AuraNoise 🎧

AuraNoise is a professional-grade color noise generator available as a **REAPER JSFX plugin** and a **responsive Web App**. 

Designed for sound designers, mix engineers, and acousticians, AuraNoise generates mathematically precise noise spectra for testing, masking, and creative synthesis.

## Features
- **8 Distinct Noise Profiles:** White, Pink, Brown (Red), Blue, Purple (Violet), Grey, Black, and Green.
- **Scientifically Accurate:** Real-time generation using digital filters—no static loops.
- **Dual-Platform:**
  - **JSFX:** Zero-latency plugin for REAPER.
  - **Web App:** Instant, browser-based generator for testing on the go.
- **Optimized Performance:** Minimal CPU usage with precise gain control.

## The Spectrum
- **White:** Equal energy per frequency.
- **Pink:** Equal energy per octave.
- **Green:** Focused at 500Hz for natural ambient masking.
- **Brown:** Deep, warm, 6dB/octave roll-off.
- **Blue/Purple:** High-frequency emphasis for "air" and surgical applications.
- **Black/Grey:** Specialized ranges for infrasonic and perceptual balance.

## Installation

### For REAPER (JSFX)
1. Open REAPER.
2. Go to `Options > Show REAPER resource path in explorer/finder`.
3. Open the `Effects` folder and create a new folder named `AuraNoise`.
4. Copy the `AuraNoise.jsfx` file into that folder.
5. Restart REAPER or press `F5` in the FX browser to refresh.
